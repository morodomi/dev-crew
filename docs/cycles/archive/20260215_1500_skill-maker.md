## Implementation Notes

### Goal
Anthropic公式ガイド(33P)の知識をフル盛りした対話型スキル作成支援。
公式skill-creatorの7つのギャップを埋める:
1. description品質（WHAT+WHEN+triggers+否定トリガー）
2. 5パターン対応
3. テスト3領域
4. トラブルシューティング
5. allowed-toolsフィールド
6. セキュリティ制約
7. バリデーションチェックリスト

### Background
Anthropic公式「The Complete Guide to Building Skills for Claude」(33P)に基づき、
公式skill-creatorの7つのギャップを埋めるスキル作成支援。
dev-crew内のスキルとして配置し、2モード（Create/Review）で動作。

### Design Approach
- SKILL.md: < 100行。モード分岐 + ステップ概要のみ
- reference.md: 公式ガイド知識のフル格納（Progressive Disclosure第3層）
- Createモード: AskUserQuestionで対話しながら段階的にスキルを構築
- Reviewモード: 既存SKILL.mdを読み取り、公式基準でチェック
- dev-crew既存スキル（init, onboard, diagnose）のパターンに準拠
